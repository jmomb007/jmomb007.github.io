Linux & GNU

Distros básicas

As distros abaixo são as distribuições recomendadas para novos usuários, geralmente semelhante ao Windows.

LOGO NOME DESCRIÇÃO
Ubuntu O Ubuntu é um sistema operacional gratuito e de código aberto, baseado no Linux, que pode ser usado em computadores e servidores.

A Canonical, uma empresa britânica, lançou o Ubuntu em 2004.
POP OS! O Pop!_OS é uma distribuição Linux gratuita e de código aberto, desenvolvida pela System76, uma fabricante de computadores Linux dos Estados Unidos.

É um dos principais concorrentes do Ubuntu no mercado de desktops, especialmente nos Estados Unidos, mas também é popular em outros países, incluindo o Brasil.
ElementaryOS elementary OS é uma distribuição de Linux baseada nos lançamentos do Ubuntu com suporte de longo prazo (LTS) e utiliza o ambiente de desktop Pantheon, que usa a linguagem de programação Vala.

A sua interface de usuário visa ser intuitiva para novos usuários sem utilizar muitos recursos.
Linux Mint O Linux Mint é um sistema operacional para computadores, projetado para funcionar na maioria dos sistemas computacionais modernos, incluindo PCs nas arquiteturas x86 e x64.

O Linux Mint cumpre as mesmas funções que os sistemas operacionais Microsoft Windows, Mac OS da Apple, e o Free BSD.

Distros intermediárias

São distros que irão demandar um conhecimento sobre GNU/Linux para poderem ser usadas propriamente.

LOGO NOME DESCRIÇÃO
Debian Debian, também conhecido como Debian GNU/Linux, é uma distribuição Linux feita de software livre e de código aberto e software proprietário desenvolvida e mantida oficialmente pelo Projeto Debian, estabelecido por Ian Murdock em 16 de agosto de 1993.
OpenSuse O openSUSE é um sistema operacional Linux gratuito e de código aberto, organizado pela comunidade e baseado no SUSE.

O openSUSE é conhecido por ser leve, mas ao mesmo tempo, oferecer uma gama de ferramentas avançadas
Fedora É um sistema operacional de código aberto e software livre, baseado na distribuição Red Hat.

O Fedora é conhecido por ser estável e seguro, além de incorporar novas tecnologias em software livre.

Ele está disponível nas versões Workstation, Server e Cloud.
EndevourOS A EndeavourOS é uma distribuição Linux baseada no Arch Linux, que oferece uma experiência Arch sem a necessidade de instalação manual.

É o sucessor do Antergos, que foi descontinuado em 2019.

Distros avançadas

São distros que demandam conhecimento avançado de Linux para serem usadas. Muitas dessas são quase totalmente customizaveis ou totalmente customizavel (LFS)

LOGO NOME DESCRIÇÃO
ArchLinux O Arch Linux é uma distribuição Linux de código aberto que se caracteriza por ser minimalista, flexível e personalizável.

Ele é conhecido por seguir o modelo "rolling release", que mantém o ambiente sempre atualizado
Gentoo O Gentoo Linux é um sistema operacional gratuito e baseado no Linux, que pode ser personalizado e otimizado de forma automática para qualquer aplicação ou necessidade.
Linux From Scratch
(LFS)
 A ideia do Linux From Scratch é instalar pacotes um a um conduzindo o usuário a um aprendizado.
Além, naturalmente, de ter que compilar todo o software especificamente para o sistema funcionar geralmente resulta em um sistema mais leve e mais rápido,
tornando assim um sistema feito por você e para você.
Slackware O Slackware Linux é um sistema operacional multitarefa, baseado em projetos de software livre, que se assemelha ao Unix.

Ele está disponível nas versões 32-bit e 64-bit e é baseado no kernel Linux 4.4 e na GNU C Library versão 2.23.
