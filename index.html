Linux & GNU

Introdução

O projeto Linux & GNU é uma iniciativa escolar que busca apresentar, de forma rápida e acessível, a história do Projeto GNU e do Kernel Linux. Através deste projeto, pretendemos oferecer uma visão concisa e clara sobre o desenvolvimento desses elementos cruciais do software livre e do código aberto.
